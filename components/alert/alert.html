Alert
Alert is a component which is used to draw user's attention to a specified thing. Tec{h}tonic has 2 categories of alert with 4 types in each.
Alert with background color

Primary! Everything looks good! Keep going...

Success! Task successful!

Alert! You might have a problem here...

Warning! Something went wrong!!!

You can add these alert by adding a common class "alert-box" and then the alert type with classes "primary", "success", "attention", "warning".

Alert with border only

Primary! Everything looks good! Keep going...

Success! Task successful!

Alert! You might have a problem here...

Warning! Something went wrong!!!

You can add these alert with by adding a common class "alert-border" and then the alert type.
